# (not) Samsung SM8250 Kernel  
### Warning:
- The kernel source code is **always** under development and may cause some unpredictable problems.
- Please use it with caution.
- **You are being warned.**

## Introduction
- This repository is based on the `lineage-23.2` branch of [LineageOS/android_kernel_samsung_sm8250](https://github.com/LineageOS/android_kernel_samsung_sm8250).

## Notes
- Code required for OneUI/ColorOS support was selectively cherry-picked from various sources.
- This kernel is combining contributions from multiple upstreams to make it as practical and well-rounded as possible.
- It supports all roms available for r8q, from android 14<sup>[1]</sup> upto 16 qpr2<sup>[2]</sup>
###### <sup>[1]</sup> It actually works fine from android11, but since people do random shit like flashing multidisabler, and blame ME that the kernel doesn't work, i just don't care about it, edit AnyKernel3 zip and use it if you need.
###### <sup>[2]</sup> I will always 'support' the latest LineageOS version available for r8q, day one.
